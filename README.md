# AI Study Library

AI Study Library is a full-stack web application designed to help students organize notes and use AI-powered tools for learning.

## Features

- User authentication using JWT
- Create and store study notes
- AI-powered note summarization
- Dashboard for managing notes
- History tracking of saved notes

## Tech Stack

Frontend
- React (Vite)
- JavaScript
- Fetch API

Backend
- Python Flask
- Flask-JWT-Extended
- SQLAlchemy ORM

Database
- SQLite

AI
- HuggingFace Transformers
